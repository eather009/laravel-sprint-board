# Security Policy

## Supported versions

Security fixes are applied to the latest `0.x` / `main` line until a stable `1.x` is released.

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Email the package owner privately via the contact details on [github.com/eather009](https://github.com/eather009), or use GitHub’s private vulnerability reporting on this repository if enabled.

Include:

- Description of the issue
- Steps to reproduce
- Affected versions
- Suggested fix (optional)

You should receive an acknowledgement within a reasonable time. Please allow time for a fix before any public disclosure.
